# 🚀 Mapa - (ReactJs - Tailwind CSS - ViteJs) 🚀

Proyecto para trabajar puntos en un mapa.

## 🛠️ Instalación

Para utilizar este proyecto primero instale las dependencias:

```bash
npm install
```

## 🕹 Uso

```bash
npm run dev
```
